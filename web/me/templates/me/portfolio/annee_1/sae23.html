Sae 23 : Mettre en place une solution informatique pour l’entreprise

Dans ce projet, nous avons eu l'opportunité de travailler sur une solution informatique impliquant à la fois le développement avec Django et la mise en place d’une infrastructure de virtualisation pour l'entreprise.

Django

Le développement Django a constitué une partie importante du projet. En binôme, nous avons dû créer cinq CRUD avec des relations entre eux. De plus, nous avons implémenté des fonctionnalités supplémentaires telles que :

  • La génération d'un récapitulatif des paramètres serveur sous forme de PDF.
  • Un système d'importation de fichiers texte avec une syntaxe particulière pour remplir automatiquement plusieurs champs dans la base de données.
  • Une mise en page soignée avec l'ajout d'un fichier CSS pour garantir une présentation visuelle agréable.

Virtualisation

Une autre partie du projet a impliqué la configuration d'une solution de virtualisation. Nous avons mis en place un serveur MySQL et configuré les tables prédéfinies nécessaires au bon fonctionnement de la base de données. Ensuite, nous avons paramétré Nginx et Gunicorn pour pouvoir afficher l'application Django sur Internet, permettant ainsi l'accès à la solution par le biais d'un serveur web sécurisé.
